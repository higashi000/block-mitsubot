# block-mitsubot

## Usage
Please set environment variable
- `SLACK_TOKEN`
- `BOT_ID`

Please regist `SLACK_TOKEN` is your slack legacy token.<br>
`BOT_ID` is your user id.<br>

```
$ git get -u github.com/higashi000/block-mitsubot

$ block-mitsubot
```
